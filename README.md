#  AI Chatbot Project

A simple chatbot built using Python that combines rule-based logic with basic NLP techniques to understand user input and respond intelligently.


##  Features

- Understands user input using multiple techniques:
  - Text cleaning (lowercase + remove symbols)
  - Keyword matching (Level 1)
  - NLP using TF-IDF & Cosine Similarity (Level 2)
  - Context awareness (Level 3)
  - Smart response selection (Level 4)

- Handles different types of conversations:
  - Greetings
  - Help requests
  - Order status
  - Jokes
  - General questions

- Provides more natural responses (Mini ChatGPT style)


##  Technologies Used

- Python  
- JSON (for chatbot intents)  
- scikit-learn (TF-IDF & Cosine Similarity)  
- Basic NLP techniques
